# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-06-05

### Added
- Initial release of JARVIS AI OS
- Core reasoning engine with 3-pass critique loop
- 10 specialized agents for diverse tasks
- Plugin SDK with hot-reload support
- Voice pipeline with wake word detection
- Memory system with semantic embedding-based recall
- WhatsApp, Telegram, Discord channel integrations
- Flutter mobile companion app with HUD interface
- Web dashboard with real-time monitoring
- Docker deployment with Docker Compose
- RAG system with ChromaDB vector store
- MCP server protocol support
- CLI interface with tab completion
