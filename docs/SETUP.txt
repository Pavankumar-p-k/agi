# JARVIS VISION AGENT — SETUP GUIDE
# 
# ── Step 1: Install Python dependencies ──────────────────────
#   pip install mss pyautogui Pillow httpx pygetwindow
#   (already have fastapi/uvicorn from existing JARVIS)
#
# ── Step 2: Make sure LLava is pulled in Ollama ──────────────
#   ollama pull llava
#   (llama3:8b and phi3:mini already needed for main brain)
#
# ── Step 3: Copy jarvis_vision_complete/ into your JARVIS dir ─
#   Place it alongside your existing jarvis_brain/ folder
#
# ── Step 4: Add 3 lines to your existing main.py ─────────────
#   (see main_integration.py below)
#
# ── Step 5: Run your server as usual ─────────────────────────
#   python main.py
#
# ── Step 6: Test it ──────────────────────────────────────────
#   curl -X POST http://localhost:8000/vision/run \
#     -H "Content-Type: application/json" \
#     -d '{"instruction": "open chrome and search for pizza near me"}'
#
# ── IMPORTANT: PC ONLY ───────────────────────────────────────
#   The Vision Agent controls your ACTUAL PC mouse and keyboard.
#   Make sure you're NOT touching the mouse while it runs.
#   Move mouse to TOP-LEFT CORNER to emergency stop (pyautogui failsafe).
#
# ── Android (optional) ───────────────────────────────────────
#   Enable USB Debugging on your phone:
#     Settings → Developer Options → USB Debugging → ON
#   Connect phone to PC via USB
#   Install ADB: https://developer.android.com/tools/adb
#   Test: adb devices  (should show your phone)
