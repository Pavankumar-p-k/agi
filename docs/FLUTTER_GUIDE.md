## Step 1 — Install Flutter

```bash
# Windows: Download Flutter SDK
# https://docs.flutter.dev/get-started/install/windows

# Add to PATH: C:\flutter\bin

# Verify
flutter doctor
# Fix any issues it shows (Android Studio, licenses, etc.)
```

## Step 2 — Set Up Android Studio

```bash
# Install Android Studio: https://developer.android.com/studio
# Open Android Studio → SDK Manager → Install Android SDK 34

# Accept licenses:
flutter doctor --android-licenses

# Enable Developer Mode on your phone:
# Settings → About Phone → tap Build Number 7 times
# Settings → Developer Options → Enable USB Debugging
```

## Step 3 — Create Flutter Project

```bash
# Create project
flutter create jarvis_app
cd jarvis_app

# Copy all the generated files into the project:
# - Replace lib/ contents with the provided files
# - Replace android/app/src/main/AndroidManifest.xml
# - Replace pubspec.yaml
```

## Step 4 — Set Up Firebase

```bash
# Install FlutterFire CLI
dart pub global activate flutterfire_cli

# Configure Firebase (run in project root)
flutterfire configure --project=your-firebase-project-id

# This generates firebase_options.dart automatically
# DELETE the placeholder firebase_options.dart and use the generated one
```

## Step 5 — Configure PC IP

```dart
// Edit lib/config/api_config.dart
// Change this line:
static const String pcIp = '192.168.1.100';
// To your actual PC IP (run `ipconfig` on Windows to find it)
```

## Step 6 — Install Dependencies

```bash
cd jarvis_app
flutter pub get
```

## Step 7 — Run the App

```bash
# Connect phone via USB (make sure USB debugging is on)
flutter devices    # Should show your phone

# Run app
flutter run

# OR build APK
flutter build apk --release
# Install: build/app/outputs/flutter-apk/app-release.apk
```

## Troubleshooting

**"Cannot connect to server"** — Make sure JARVIS Python server is running on PC. Check IP address in api_config.dart matches your PC.

**"Firebase auth error"** — Replace firebase_options.dart with the one generated by `flutterfire configure`.

**Camera permission denied** — Go to phone Settings → Apps → JARVIS → Permissions → Allow Camera.

**App crashes on start** — Run `flutter clean && flutter pub get`, then try again.

**Google Fonts not loading** — Requires internet connection on first load. Fonts are cached afterwards.
