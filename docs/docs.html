Quick Start

# Install & run
$ git clone https://github.com/prasanthibanavath2008-blip/jarvis
$ cd jarvis
$ pip install -e .
$ jarvis server
# In another terminal:
$ jarvis cli

CLI Commands

/website

Generate a multi-page website from a topic

/build

Start an autonomous build with control loop

/supervisor

Launch multi-agent autonomous build

/generate-ui

Generate a UI from template description

/plan

Preview an execution plan

/goal

Submit a long-running goal

Architecture

JARVIS uses a modular architecture with FastAPI backend, tiered memory, plugin system, governance layer, and multiple frontends (CLI, Web, Flutter). All core processing runs locally with optional cloud augmentation.
